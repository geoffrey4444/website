Workshop on Gravitational Waves and High Performance Computing 2022

California State University, Fullerton

These are materials from the 2022 Cal State Fullerton Workshop on High-Performance Computing and Gravitational Waves.

The workshop is scheduled to be held at Cal State Fullerton, hosted by Nicholas and Lee Begovich Center for Gravitational-Wave Physics and Astronomy, on August 15, 2022 – August 19, 2022. The workshop is planned to be held in person in McCarthy Hall room 629 (campus map), but if necessary because of COVID-19, we will switch to an online format.

Applying

To apply for the 2022 workshop, please contact Marianne Smith. If you have questions about the workshop, please don’t hesitate to contact me.

Links

  • Notebook to explore data from SXS Catalog of simulations of merging black holes - This notebook will let you interact with data from different simulations in the SXS Catalog. It will show you how to make plots of the black-hole masses, and it will show you how to plot and turn into a sound the emitted gravitational waves.

  • cocalc.com - Run Python notebooks collaboratively for a monthly fee
  • colab.research.google.com - Run Python notebooks on Google’s supercomputers for free
  • mybinder.org - Run Python notebooks that are already uploaded to github.com and also UNIX terminals for free
  • PyCBC Tutorials - Tutorials introducing gravitational-wave data analysis using Python notebooks
  • LIGO Open Science Center - Learn about LIGO science, including playing with real LIGO data using python
  • ParaView - Software to visualize 3D data
